# API документация для math::Matrix

Этот документ описывает публичный интерфейс шаблонного класса `math::Matrix<T, Rows, Cols, Allocator>`.

## Общие сведения

Класс `math::Matrix` поддерживает:
- статическую и динамическую размерность (`Rows`, `Cols`, `Dynamic`)
- арифметические операции
- матричное умножение
- транспонирование, след, детерминант и обратную матрицу
- ввод/вывод в текстовом и бинарном форматах

### Параметры шаблона

```cpp
template<
    Arithmetic T,
    std::size_t Rows = Dynamic,
    std::size_t Cols = Dynamic,
    typename Allocator = std::pmr::polymorphic_allocator<T>
>
class Matrix;
```

- `T` — арифметический тип (`int`, `double`, `float` и др.)
- `Rows`, `Cols` — размеры матрицы, или `Dynamic` для динамической размерности
- `Allocator` — аллокатор для динамической матрицы

## Конструкторы

### `Matrix()`

- Статические матрицы создаются фиксированного размера и заполняются нулями.
- Динамические матрицы создаются размером `0x0`.

### `Matrix(size_type rows, size_type cols)`

- Доступен только для динамических матриц.
- Выделяет память и заполняет все элементы значением `T{}`.

### `Matrix(std::initializer_list<std::initializer_list<T>> init)`

- Создаёт матрицу из вложенного списка.
- Проверяет согласованность размеров строк.
- Поддерживает статические и динамические матрицы.

### Копирующие и конвертирующие конструкторы

- `Matrix(const Matrix&)` и `Matrix(Matrix&&)` — стандартное копирование/перемещение.
- `Matrix(const Matrix<U, R2, C2, Alloc2>& other)` — конвертация между совместимыми типами.
- `Matrix(const E& expression)` — конструктор из выражения, поддерживающего концепт `MatrixExpression`.

## Доступ к элементам

### `operator()(size_type i, size_type j)`

- Прямой доступ к элементу в строке `i` и столбце `j`.
- Используется как для чтения, так и для записи.

### `operator[](size_type i)`

- Возвращает прокси-строку, что позволяет обращаться к элементам как `m[i][j]`.

### `rows()`, `cols()`, `size()`, `data()`

- `rows()` и `cols()` возвращают текущие размеры матрицы.
- `size()` возвращает общее количество элементов.
- `data()` возвращает указатель на внутренний буфер в row-major порядке.

## Итераторы

- `begin()`, `end()`, `cbegin()`, `cend()` доступны для обхода всех элементов.
- `rbegin()`, `rend()` поддерживают обратный порядок.

## Статические фабрики

### `zeros()` / `zeros(rows, cols)`

Создаёт матрицу, заполненную нулями.

### `ones()` / `ones(rows, cols)`

Создаёт матрицу, заполненную единицами.

### `fill(value)` / `fill(rows, cols, value)`

Заполняет матрицу указанным значением.

### `identity()` / `identity(n)`

Создаёт единичную матрицу.

### `from_diag(std::initializer_list<T> diag)`

Создаёт диагональную матрицу из списка значений диагонали.

### `from_diag(size_type n, const T& value)`

Для динамических матриц создаёт диагональную матрицу размера `n x n`, заполненную одним значением на диагонали.

## Математические операции

### Поэлементные операции

- `operator+=`, `operator-=`, `operator*= (scalar)`, `operator/= (scalar)`
- оператор `+`, `-`, `*` со скаляром и `operator/` со скаляром

### Матричное умножение

- `operator*(const Matrix& other)` выполняет обычное матричное умножение.
- длины строк и столбцов проверяются на совместимость.

## Линейная алгебра

### `transpose()`

Возвращает транспонированную матрицу.

### `trace()`

Вычисляет сумму диагональных элементов.
Бросает `std::logic_error`, если матрица не квадратная.

### `determinant()`

Вычисляет детерминант квадратной матрицы.
Бросает `std::logic_error`, если матрица не квадратная.

### `inverse()`

Возвращает обратную матрицу.
Бросает `std::logic_error`, если матрица не квадратная.

## Сравнение

- `operator==` сравнивает две матрицы по размеру и значениям элементов.
- Для плавающих типов используется сравнение с точностью `epsilon`.
- `operator<=>` выполняет лексикографическое сравнение по размеру и элементам.

## Ввод/вывод

### `operator<<`

Форматирует матрицу в текстовом виде: по одной строке на каждый ряд, значения разделяются пробелами.

### `operator>>`

Читает матрицу из текстового представления.

### `write_binary` / `read_binary`

- `write_binary(std::ostream&)` сохраняет размеры и данные в бинарном формате.
- `read_binary(std::istream&)` загружает матрицу из бинарного формата.
- Для статических матриц размеры должны совпадать с фиксированными параметрами.

## Пример использования

```cpp
#include <iostream>
#include "matrix.hpp"

int main() {
    using math::Matrix;
    Matrix<double> a = Matrix<double>::identity(3);
    Matrix<double> b = Matrix<double>::fill(3, 3, 2.0);
    Matrix<double> c = a * b;

    std::cout << "Matrix A:\n" << a << "\n\n";
    std::cout << "Matrix B:\n" << b << "\n\n";
    std::cout << "Matrix C = A * B:\n" << c << "\n";

    return 0;
}
```

## Полезные замечания

- Для статических матриц проверка размеров выполняется на этапе компиляции.
- Для динамических матриц размеры задаются во время выполнения.
- Стандартный порядок хранения данных — row-major.

## Рекомендуемые сценарии

- `Matrix<double, 3, 3>` — для матриц фиксированного размера.
- `Matrix<float>` — для матриц произвольного размера во время выполнения.
- `Matrix<int, 4, Dynamic>` — для матриц со статическим числом строк и динамическим числом столбцов.
